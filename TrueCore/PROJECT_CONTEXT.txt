PROJECT: TrueCore Packet Assistant

Purpose:
Desktop Python tool that audits VA medical packet submissions.

System Overview:
TrueCore analyzes medical packet submissions from provider offices and determines whether the packet meets VA documentation requirements.
It detects forms, extracts fields, validates ICD codes, scores packet completeness, and produces structured results for review.

Main Architecture:
Launcher → Engine → GUI → Packet Processing Pipeline

Core Processing Flow:
GUI → packet_processor → parser → extractor → validation

Primary Components:

Launcher System
PySide6 launcher responsible for update checks, engine download, and starting the TrueCore engine.

Engine
Main executable containing the TrueCore processing system and GUI.

GUI System (PySide6)
Modern PySide6 interface used for packet auditing, file selection, results display, and export tools.

Processing Pipeline
Packet analysis engine that parses documents, extracts fields, validates data, and scores packet completeness.

---

Core Files:

truecore_app.py
Engine entrypoint. Initializes runtime environment and launches the PySide6 GUI.

pyside_app.py
Starts the PySide6 application and loads the main GUI window.

main_window.py
Main GUI interface displaying packet audit results, console logs, packet details, and administrative tools.

launcher_app.py
Entry point for the TrueCore launcher system.

launcher_window.py
PySide6 interface for the launcher including login area, update status, and engine launch control.

updater.py
Handles update checks, downloads engine releases, and installs updates.

---

Processing Engine Files:

packet_processor.py
Main audit engine. Handles packet scoring, issue detection, form identification, and suggestion generation.

parser.py
Extracts raw text from supported document types including PDF, DOCX, and images.

extractor.py
Pulls structured fields and patient data from parsed document text.

packet_classifier.py
Classifies packet types and determines document roles within a submission.

packet_triage.py
Handles packet routing logic and determines audit outcomes.

form_detector.py
Detects specific medical forms within packet documents.

validator.py
Validates extracted fields and checks packet completeness.

suggestions.py
Generates suggested fixes for missing or incomplete packet information.
