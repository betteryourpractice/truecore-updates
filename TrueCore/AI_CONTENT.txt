TRUECORE AI ENGINEERING CONTEXT

Owner: TrueBrain™, LLC

for VA Community Care requests. It performs document parsing, field extraction,
form detection, validation scoring, and packet diagnostics.

The system includes:

• GUI application
• packet analysis engine
• OCR parsing
• ICD detection
• form detection
• packet validation scoring
• packet suggestions engine
• administrative monitoring panel
• logging and audit system
• executable build system

AI may assist with maintenance, debugging, and controlled upgrades.

----------------------------------------
TRUECORE PROCESSING PIPELINE DETAILS
----------------------------------------

TrueCore processes packets through a deterministic multi-stage pipeline.

The stages are executed in the following order:

1. Document Parsing
   Module: parser.py

   Responsibilities:
   • Detect document type
   • Extract text from PDF, DOCX, images, or TXT
   • Apply OCR preprocessing for scanned documents

2. Text Normalization
   Module: text_normalizer.py

   Responsibilities:
   • Normalize whitespace
   • Repair OCR header splits
   • Standardize line structure

3. Field Extraction
   Module: extractor.py

   Responsibilities:
   • Extract structured fields including:
     - patient_name
     - dob
     - authorization_number
     - icd_codes
     - ordering_doctor
     - referring_doctor

4. ICD Detection
   Module: icd_lookup.py

   Responsibilities:
   • Detect diagnosis codes using:
     - regex scanning
     - contextual detection
     - clinical inference

5. Form Detection
   Module: form_detector.py

   Responsibilities:
   • Detect packet forms using keyword fingerprints
   • Apply probability scoring for form likelihood

6. Section Parsing
   Module: section_parser.py

   Responsibilities:
   • Split documents into logical form sections
   • Detect clinical segments such as:
     - diagnosis
     - procedure
     - provider
     - authorization

7. Document Intelligence
   Module: document_intelligence.py

   Responsibilities:
   • Detect tables
   • Detect checkboxes
   • Detect provider signatures
   • Identify structured clinical sections
   • Extract semantic clinical narratives

8. Packet Validation
   Module: validator.py

   Responsibilities:
   • Validate extracted packet fields
   • Verify required forms exist
   • Generate validation issues
   • Calculate packet score

9. Suggestion Engine
   Module: suggestions.py

   Responsibilities:
   • Convert issues into correction instructions
   • Generate remediation guidance
   • Provide TrueBrain diagnostic summaries

10. Packet Result Return
    Module: packet_processor.py

    Responsibilities:
    • Combine extracted data
    • Attach validation results
    • Return structured packet audit result to GUI

----------------------------------------
PACKET SCORING MODEL
----------------------------------------

TrueCore scoring begins at 100 and deducts points for validation failures.

Example deductions:

Missing patient name        → -20
Missing authorization       → -20
Missing DOB                 → -15
Missing ICD codes           → -15
Missing ordering provider   → -10
Missing referring provider  → -10
Missing SEOC                → -5
Missing LOMN                → -5
Missing consultation form   → -5

Score ranges determine packet confidence:

90-100 → HIGH
70-89  → MEDIUM
0-69   → LOW
